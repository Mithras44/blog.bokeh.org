The Bokeh Ecosystem

With a core library (and multi-lang support), Bokeh is a Python interactive visualization library that targets modern web browsers for presentation.

  • Bokeh: Core library
  • Multi-lang Support
    • rBokeh: R Bindings
    • Bokeh-Scala: Scala Bindings
    • Bokeh.jl: Julia Bindings
  • Datashader: a graphics pipeline system for creating meaningful representations of large amounts of data
